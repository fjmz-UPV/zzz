# Repositorio de resultados

En la presente proyecto entran en juego dos repositorios: el de **resultados** y el del **enunciado**.

Este es el repositorio (con permisos de lectura y escritura exclusivos y esta vez compartidos por todos los integrantes del grupo) de **resultados** del proyecto del Seminario de "Control de versiones con Git y GitHub". Antes de clonar este repositorio debe leerse el enunciado de la práctica cuyo enlace se muestra a continuación: [https://github.com/MUTSRC-GIT-25-26/Trabajo-Enunciado.git](https://github.com/MUTSRC-GIT-25-26/Trabajo-Enunciado.git)

Llegado el caso, el proceso de clonación fue ya detallado en la práctica anterior. El contenido del directorio versionado una vez clonado debe ser el siguiente:

```text
<DIR_RESULTADOS_PRACTICA_2>
├── .git
├── .gitignore
├── README.md
├── figuras
├── versos1.txt
├── versos2.txt
├── versos3.txt
├── versos4.txt
├── estrofa.txt
├── estrofa_objetivo.txt
└── roles.txt
```

En el transcurso de la práctica, los resultados requeridos se deben dejar en este repositorio compartido por todos los integrantes del grupo de la manera indicada en el enunciado de la práctica.
